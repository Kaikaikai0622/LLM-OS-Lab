# Benchmark Tasks v3 - 三模式对比实验
# 7 个任务：低输出(2) + 中输出(4) + 高输出(1)
# 目标：验证 CM 模式在多轮工具调用后有效降低 context token 规模

# ═══════════════════════════════════════════════════════
#  低输出任务 × 2（基准对照，预期 1-2 轮）
# ═══════════════════════════════════════════════════════

# Task 1 - Fibonacci（低输出）
计算 Fibonacci 数列第 30 项的值，以及 F(30)/F(29) 的比值，返回结果保留10位小数。不需要打印整个数列。

# Task 2 - 素数统计（低输出）
用埃拉托色尼筛法求 1-500 内所有素数，返回：素数总个数、最大的5个素数、所有素数之和。不需要打印每轮筛除过程。

# ═══════════════════════════════════════════════════════
#  中输出任务 × 4（核心差异区间，预期 3-7 轮）
# ═══════════════════════════════════════════════════════

# Task 3 - 高斯消元过程（中输出，多步输出）
生成一个 5x6 增广矩阵(整数 1-20)，执行高斯消元求解。打印：初始矩阵（5行）、每一步的主元和消元后的矩阵（最多3步）、最终解向量。控制输出在30行以内。

# Task 4 - 排序算法对比（中输出，多轮修错概率高）
实现快速排序和归并排序，对20个随机整数(范围1-100)排序，打印每趟排序的中间状态、最终的比较次数和交换次数统计对比表。

# Task 5 - 迷宫求解（中输出，BFS 修错概率高）
随机生成 8x8 迷宫(0=通路,1=墙壁，确保起点(0,0)到终点(7,7)可达)，用 BFS 求最短路径。打印迷宫地图、探索的节点数、最短路径长度和路径坐标。

# Task 6 - 积分收敛分析（中输出，多步计算）
用梯形法则对 sin(x)·ln(x+1) 在 [0,5] 上积分，分别取 n=10,50,200,1000 四个区间数。对每个 n 打印：区间数、积分结果(保留8位小数)、与上一个 n 的绝对误差。最后给出收敛趋势总结。

# ═══════════════════════════════════════════════════════
#  高输出任务 × 1（压力测试，预期 3-6 轮）
# ═══════════════════════════════════════════════════════

# Task 7 - 统计模拟（高输出，长输出压力测试）
用纯Python(不用numpy)生成100个正态分布随机数(均值50,标准差15)，计算描述统计量(均值、中位数、标准差、最小值、最大值、四分位数)，绘制文本直方图(20个bin，用*号表示频率)，并用均值±2σ规则检测异常值。打印完整统计表和直方图。
