# Benchmark Tasks Subset 5 - v2
# 任务类型分布：低输出(3个) + 中输出(2个)
# 适配 summary_max_chars=2000 的 Context Mode 测试

# ═══════════════════════════════════════════════════════
#  低输出任务 × 3 (预期输出 < 200 chars)
# ═══════════════════════════════════════════════════════

# Task 1 - Fibonacci 计算（低输出）
计算 Fibonacci 数列第 30 项的值，以及 F(30)/F(29) 的比值，返回结果保留10位小数。不需要打印整个数列。

# Task 2 - 矩阵特征值统计（低输出）
生成一个 10x10 随机对称矩阵(整数 1-20)，计算并返回：最大特征值、最小特征值、特征值之和。不需要打印矩阵和特征向量。

# Task 3 - 素数统计（低输出）
用埃拉托色尼筛法求 1-500 内所有素数，返回：素数总个数、最大的5个素数、所有素数之和。不需要打印每轮筛除过程。

# ═══════════════════════════════════════════════════════
#  中输出任务 × 2 (预期输出 800-1500 chars)
# ═══════════════════════════════════════════════════════

# Task 4 - 高斯消元过程（中输出）
生成一个 5x6 增广矩阵(整数 1-20)，执行高斯消元求解。打印：初始矩阵（5行）、每一步的主元和消元后的矩阵（最多3步）、最终解向量。控制输出在30行以内。

# Task 5 - 积分收敛分析（中输出）
用梯形法则对 sin(x)·ln(x+1) 在 [0, 5] 上积分，分别取 n=10,50,200 三个区间数。对每个 n 打印：区间数、积分结果、与上一个 n 的误差。不需要打印每个梯形的面积。
